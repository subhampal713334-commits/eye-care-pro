**👁️ AI EyeCare Pro+ — Intelligent Retinal Disease Detection System**

AI EyeCare Pro+ is a deep learning–powered clinical decision support system designed to assist in the early detection and analysis of retinal diseases using OCT (Optical Coherence Tomography) scans. The system leverages a MobileNetV2-based convolutional neural network to classify retinal images into four categories: CNV, DME, DRUSEN, and NORMAL.

Beyond basic prediction, the platform integrates explainable AI through Grad-CAM, enabling visualization of regions in the scan that influence the model’s decision. This enhances transparency and builds trust in AI-assisted diagnosis.

The system also generates a structured clinical report that includes diagnosis, confidence score, medical explanation, risk assessment, recommended treatment plans, and follow-up guidelines—bridging the gap between AI prediction and real-world medical usability.

With a modern, user-friendly interface built using Streamlit, AI EyeCare Pro+ provides real-time inference, making it suitable for rapid screening, academic research, and assistive healthcare applications.

**Key Features:**

* Deep Learning-based retinal disease classification
* Explainable AI with Grad-CAM visualization
* Confidence-aware predictions
* Automated clinical report generation
* Modern, intuitive UI for seamless interaction

AI EyeCare Pro+ aims to support ophthalmologists and improve accessibility to early eye disease detection, especially in resource-limited settings.# eye-care-pro
